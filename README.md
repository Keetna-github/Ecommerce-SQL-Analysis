# E-commerce SQL Analysis

## Project Overview
SQL Analysis of Brazilian E-commerce dataset
using MySQL with 18 queries!!

## Database Tables
- customers (99,441 records)
- orders (96,476 records)
- payments (100,754 records)
- reviews (270 records)

- SELECT, WHERE, GROUP BY, ORDER BY
- JOINS (Single and Multiple)
- Subqueries
- Window Functions (OVER)
- CASE Statement
- Aggregate Functions (SUM, AVG, COUNT)
- Date Functions (DATEDIFF)
- HAVING clause
- Views
- Stored Procedures
